## What is Anon?

Anon is a dark-style eCommerce frontend template. It covers the core pages of an online store — product grid, header with navigation, and a cart icon — with a clean, modern aesthetic. No frameworks, no dependencies.

---

## Features

- **Dark color scheme** — high-contrast layout built around a dark palette
- **Product grid** — responsive card layout for listing items
- **Navigation header** — logo, links, and cart icon included
- **Fully responsive** — scales cleanly from mobile to desktop
- **No build step** — open the HTML file and it works

---

## How to Use

```bash
# Windows
start index.html

# macOS
open index.html
```

Or use the **Live Server** extension in VS Code: right-click `index.html` → *Open with Live Server*.
